структура проекта:
excoursionsTgbot2/
├─ common/ #  константы и настройки, используемые по всему проекту
├─ database/ # Папка для всего связанного с базой данных: подключение, модели, сессии и конфигурация БД
│  └─ orm_queries/ # Пакет с запросами к базе данных: функции для выборок и работы с ORM запросами
├─ filters/ # Пользовательские фильтры для апдейтов (фильтр по роли пользователя и типа чата (групповой/личный))
├─ handlers/ # Обработчики событий и команд бота: хендлеры сообщений, коллбэков
├─ kbds/ # Клавиатуры: инлайн-клавиатуры, Reply-клавиатуры, кнопки и их компоновка для интерфейса бота
├─ middlewares/ # Промежуточные слои для обработки обновлений: сессия базы данных
└─ utils/ # Вспомогательные функции
